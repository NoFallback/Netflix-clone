# Netflix Clone
## Project Overview 
This project is a Netflix Login Page Clone created using HTML and CSS as a frontend practice exercise. The goal was to improve my understanding of webpage structure, styling, layouts, responsiveness, and recreating real-world user interfaces.

While it is not a pixel-perfect copy of Netflix's original login page, it closely follows the overall design and layout for learning purposes.

## Features
1. Modern Netflix-inspired UI
2. Responsive layout
3. Styled login form
4. Background image with overlay effect
5. Clean and organized HTML & CSS structure

## Getting Started

Follow these steps to run the project locally on your machine.

### 1. Clone the Repository
git clone https://github.com/NoFallback/Netflix-clone.git
### 2. Navigate to the Project Folder
cd Netflix-clone
### 3. Open the Project

You can simply open the index.html file in your browser.

### Option A: Double Click

Open index.html directly.

### Option B: Using VS Code Live Server (Recommended)
Open the project folder in VS Code.
Install the Live Server extension.
Right-click on index.html.
Click Open with Live Server.

The project will be available at: http://127.0.0.1:5500 or a similar localhost address generated by Live Server.
